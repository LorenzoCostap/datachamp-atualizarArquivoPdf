# Importar Arquivo XLS e Gerar Bloco K (Jomat)

## Passo a Passo

1. Baixar o arquivo `.XLS` recebido por e-mail.

2. Abrir o arquivo padrão:

```
G:\jomat\padrão de importação.xls
```

3. Copiar do arquivo recebido as colunas:
- referência;
- Quantidade.

4. No Sentus, acessar:

```
Jomat → Administrativo → Importador
```

5. Preencher os campos:

| Campo | Valor |
|--------|--------|
| Tipo de Arquivo | Bloco K - XLS |
| Data de Apuração | Último dia do mês anterior |
| Arquivo | Upload do XLS |

6. Clicar em **Gerar K**.

7. Salvar o arquivo gerado.

8. Enviar o arquivo por e-mail.

# Séries de Documentos

## Objetivo

Definir quais documentos fiscais o emitente poderá emitir.

## Passo a Passo

1. Acesse:

   ```
   Administrativo → Série de Documentos
   ```
2. Clique em **Novo**.

3. Selecione o emitente.

4. Escolha o documento:

 - 55 - NF-e
 - 57 - CT-e
 - 58 - MDF-e
 - 65 - NFC-e

5. Informe:
  - Série;
  - Número.

### Observação

- Caso o cliente nunca tenha emitido documentos:
  - Série: **1**
  - Número: **0**

- Caso já tenha emitido:
  - Informar o número correspondente ao último documento emitido.
